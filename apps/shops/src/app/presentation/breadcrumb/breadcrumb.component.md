# Breadcrumb Component

Render breadcrumb with design of shops.

---

## Usage

`html`

```html
<shops-breadcrumb
  [breads]="categories"
></shops-breadcrumb>
```

## Properties

| breads      |                                |
| ----------- | ------------------------------ |
| Description | Set list of strings for breads |
| Attribute   | breads                         |
| Type        | `string[]`                     |
| Default     | `[]`                           |
